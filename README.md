# 💰 Zorvyn Finance Dashboard
A clean, interactive, and responsive finance management dashboard built with **React.js**. This project allows users to track their financial activities, visualize spending patterns, and manage transactions with role-based access.

## ✨ Features
* **Financial Summary:** Real-time calculation of Total Balance, Monthly Income, and Total Expenses.
* **Interactive Charts:** * **Balance Trend:** Time-based line chart showing balance fluctuations.
    * **Spending Breakdown:** Categorical pie chart for expense analysis.
* **Transaction Management:**
    * Detailed list showing Date, Category, and Amount.
    * **Search & Filter:** Quickly find transactions by category.
    * **Simulated CRUD:** Ability to edit transactions in Admin mode.
* **Role-Based UI (RBAC):** * **Admin Mode:** Full access to view and edit transactions.
    * **Viewer Mode:** Read-only access for secure data viewing.
* **Dark Theme UI:** Optimized for better readability of financial data.
* **Fully Responsive:** Works seamlessly on Desktop, Tablet, and Mobile.

## 🛠️ Tech Stack
* **Frontend Library:** React.js
* **Data Visualization:** Recharts
* **Icons:** Lucide-React
* **State Management:** React Hooks (`useState`, `useMemo`)
* **Styling:** CSS3 / Modern Layouts
